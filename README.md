# ey-hackathon

EY Hackathon Covid Efficacy Tracker

## Installation
```
$ git clone git@github.com:DaKeiser/ey-hackathon.git
```

## Setup
To run this project, follow these steps

```
$ cd ey-hackathon
$ pipenv install
$ pipenv python3 app.py
```

To run using docker

```sh
$ docker-compose build
$ docker-compose up
```
